Sobre

Dedicatória: Projeto de dados abertos, motivado pela eterna memória de Aaron Swartz.

O Economia Popular é um projeto de exploração e agregação de Dados Abertos que nasceu visando dar informação estatística de forma clara, objetiva e atualizada para o povo brasileiro.

Nossa ideia é agregar diversos índices, séries e métricas atualizadas a partir de suas fontes oficiais diariamente, de modo a facilitar o acesso a esses dados de forma objetiva, e deixar claro o real estado da saúde econômica do nosso país, de modo a contribuir para combate de Fake News, manipulações estatísticas na internet e na grande mídia.

O projeto se baseia em diversas técnicas de coleta, tratamento, análises estatísticas, machine learning e correlação de dados, afim de disponibilizar os mesmos em formatos de séries históricas para a grande população, e também para outros economistas, estatísticos e cientistas de dados.

Todos os nossos índices contém links para download dos dados apresentados em diversos formatos, as reais fontes citadas e periodicamente atualizamos os datasets no Kaggle para análise exploratória aberta.

Ele foi criado por Matheus Fidelis no dia 2 de dezembro de 2022, fim de resolver o problema de dispersão de índices e fontes para facilitar seu acompanhamento e análises macroeconômicas para pesquisas acadêmicas na área de Economia, Ciencia de Dados e Analytics. Porém, a ideia se mostrou promissora para outras pessoa e decidiu-se abrir para o grande público para cumprir um papel social maior.

Matheus Fidelis é Engenheiro de Confiabilidade (SRE), Desenvolvedor Especialista em Tecnologias Cloud Native e entusiasta de Politica, Economia, Ciência de Dados, Machine Learning e Dados Públicos.

Se mantém ativo na comunidade de Software gerando conteúdos em forma de Artigos, Vídeos e diversos projetos de Código Aberto. É apoiador de projetos sociais e ativista do Open Source.
